Concepto  SQL  Equivalente  en  Django  ORM  
Tabla  Clase  que  hereda  de  models.Model  
Columna  /  Campo  Atributo  de  clase  con  tipo  de  campo  (CharField,  IntegerField…)  
Fila  /  Registro  Instancia  del  modelo  
Clave  primaria  Campo  con  primary_key=True  (Django  crea  id  automático)  
Clave  foránea  models.ForeignKey(Modelo,  on_delete=…)

SELECT  Modelo.objects.all()  /  ﬁlter()  /  get()

UPDATE  instancia.save()  o  queryset.update()  
DELETE  instancia.delete()  o  queryset.delete()
